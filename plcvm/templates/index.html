About

Virtual Microscope is a Python/Django application to view and annotate very high-resolution images.

A virtual microscope relieves pressure on laboratories and equipment, and makes it possible to to collaborate remotely - without needing to transport precious slides or colleagues.

It was created at New York University School of Medicine for teaching and research purposes but the basic principle is suitable for exploring any high-resolution images - from astronomy, museum collections, etc.

NYU's application was open-sourced but has languished; Cardiff University School of Medicine and NHS hackday have brought the project up-to-date, packaged it and added documentation, making it easier to get started with and more attractive to developers.

Virtual Microscope is hosted on GitHub.

This particular implementation is by the UCT Pathology Learning Centre and should be considered experimental. (It may have bugs!)

The Virtual Microscope allows both learners and faculty to annotate slides with community peer-review. It includes a practical exam system, and each slide can include metadata from controlled vocabularies to classify and search for slides. We have released the source code of the application as an open-source project on Google Code.
